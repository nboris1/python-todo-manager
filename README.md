## 🛠️ Project 1: CLI Todo List Manager
A clean, command-line interface (CLI) task manager built in Python using core programming logic and loops.

### ✨ Current Features
- **Dynamic Task Addition:** Add as many tasks as you need on the fly.
- **Indexed Viewing:** Displays tasks in an organized, numbered list.
- **Loop-Driven Interface:** Runs continuously until you choose to exit.

### 🚀 Future Roadmap
- [ ] Add a feature to safely remove completed tasks
- [ ] Save tasks to a text file so they don't disappear when the program closes
- [ ] Add categories or priority tags to tasks
